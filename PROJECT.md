# Project Overview

## Goal
Build a simple full-stack image upload and preview web application.

## Tech Stack
Frontend: React + Tailwind CSS
Backend: Node.js + Express.js
API: REST

## Core Features
- Home / Landing page
- Image upload (.jpg, .png only)
- Gallery / Uploads list
- Image preview modal

## Constraints
- Must follow defined screen workflow
- Only React + Tailwind frontend
- Only Node + Express backend
- REST API communication
- File type validation (.jpg, .png)

## Definition of Done
Users can:
- Navigate between screens
- Upload images
- View uploaded images
- Preview images in modal
